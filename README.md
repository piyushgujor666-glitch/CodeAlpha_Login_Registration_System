# 🔐 Login & Registration System (CodeAlpha C++ Internship)

## 📌 Project Overview

The **Login & Registration System** is a C++ console application developed for the **CodeAlpha C++ Programming Internship**. It allows users to register with a username and password, stores user credentials in files, and verifies users during login.

## ✨ Features

* User Registration
* Duplicate Username Check
* Secure File-Based Storage
* User Login Authentication
* Success and Error Messages
* Menu-Driven Interface

## 🛠️ Technologies Used

* C++
* File Handling
* Functions
* String Handling

## 📂 Project Structure

```
CodeAlpha_Login_Registration_System/
│── main.cpp
│── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project in any C++ IDE.
3. Compile the `main.cpp` file.
4. Run the executable.
5. Register a new user and log in using the created credentials.
